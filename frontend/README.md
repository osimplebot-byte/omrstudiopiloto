# Frontend Omr Studio Piloto

## Requisitos
- Node.js 20+
- pnpm 8+

## Instalação
```bash
pnpm install
```

## Desenvolvimento
```bash
pnpm dev
```

## Build e preview
```bash
pnpm build
pnpm preview
```

Configure variáveis em `.env` seguindo `.env.example`. As integrações com Supabase e N8N usam apenas URLs públicas; nunca commit secrets.
